# 万维银客城 (INKE CITY)

一个赛博朋克风格的阅读社区网站，融合作品分享和社区交流功能。

## 🎨 设计特色

- **赛博朋克主题**：霓虹紫、霓虹粉、霓虹蓝配色
- **炫酷特效**：发光效果、扫描线、粒子动画
- **响应式设计**：完美支持移动端、平板和桌面端
- **现代UI**：使用Orbitron和Rajdhani字体

## ✨ 核心功能

### 1. 用户系统
- ✅ 用户注册/登录（Supabase Auth）
- ✅ 用户个人主页
- ✅ 用户资料展示

### 2. 作品库（Works Library）
- ✅ 作品上传（标题、简介、内容、封面）
- ✅ 封面图片上传（Supabase Storage）
- ✅ 标签系统（多标签支持）
- ✅ 作品列表展示（网格布局）
- ✅ 作品详情页面
- ✅ 按标签筛选

### 3. 闲聊版面（Chat Forum）
- ✅ 发帖功能
- ✅ 回复/评论功能
- ✅ 帖子列表展示
- ✅ 帖子详情页

### 4. 搜索功能
- ✅ 按作者搜索
- ✅ 按作品名称搜索
- ✅ 按标签搜索
- ✅ 多类型筛选（作品/帖子/作者/标签）

## 🛠️ 技术栈

### 前端
- **React 18** + **TypeScript**
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **React Router** - 路由管理
- **Lucide React** - SVG图标库

### 后端
- **Supabase Auth** - 用户认证
- **Supabase Database** - PostgreSQL数据库
- **Supabase Storage** - 图片存储

## 📝 功能使用指南

### 注册与登录
1. 访问网站首页
2. 点击右上角"注册"按钮
3. 填写邮箱、用户名、显示名称和密码
4. 或使用测试账号直接登录

### 发布作品
1. 登录账号后，点击导航栏"作品库"
2. 点击"发布作品"按钮
3. 填写标题、简介、内容
4. 上传封面图片（可选）
5. 添加标签（可选）
6. 点击"发布作品"

### 发表帖子
1. 登录后，访问"闲聊版面"
2. 点击"发新帖"按钮
3. 选择分类（闲聊/通用）
4. 填写标题和内容
5. 点击"发布帖子"

### 搜索功能
1. 点击导航栏"搜索"
2. 输入关键词
3. 选择搜索类型（全部/作品/帖子/作者/标签）
4. 点击"搜索"按钮查看结果
